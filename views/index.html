Objetivo:

El objetivo de esta pagina y/o proyecto es el tratado de datos en formato json, para mostrar los datos de manera que este los pueda entender de una manera mas sencilla, en la primera parte de este proyecto se trabajara con la base de un archivo json, proporcionado por el gobierno del estado de veracruz, en donde se almacenan algunas ubicaciones de sus vehiculos de recoleccion de basura

En la segunda fase de este proyecto permitira el trabajo con base de datos, en este punto se maneja informacion real obtenida de el Municipio de Metepec Hidalgo, en donde se ubican algunas comunidades del Municipio con el fin de obtener la ubicacion de estas
